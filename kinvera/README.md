# Kinvera Website

A simple landing page for Kinvera, a trusted home care network connecting homeowners with vetted Home Stewards.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project works well on Vercel, Netlify, or GitHub Pages.

## Main files

- `src/App.jsx` — website content and structure
- `src/styles.css` — visual styling
- `index.html` — page metadata
- `package.json` — dependencies and scripts
